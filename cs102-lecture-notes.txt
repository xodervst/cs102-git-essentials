git status - To check status of the current project.
git add - to stage the files
git log - to view the commit history
git init - to initialise the empty repository
git push - to push the changes from the server

Git is a open source version control system and it enables you to store code, track revision history, merge code changes and revert to earlier code versions whenever we need. 
Git stores the source code and all the history of development and we can create a copy of our source code known as a branch. We can work in parallel to the main version of the source code. After confirming the changes, we can merge it with the main version.

Markdown - It is a markup language to create formatted texts using a plain text editor.
